# START HERE

This public repo is the free starter pack for Lenny's Data.

## Repo guide
- `index.json`: source-of-truth metadata index for the sample dataset
- `newsletters/`: 10 curated newsletter markdown posts
- `podcasts/`: 20 podcast markdown transcripts
- `LICENSE.md`: starter-pack usage terms

## Included in this repo
- Newsletter posts: 10
- Podcast episodes: 20
